Get started with Elfsight Weather 1.2.0

Installation

1. Install Elfsight Weather WordPress Plugin elfsight-weather-cc.zip to your website. Don't know how? Please learn here.

2. Go to Weather page in WordPress admin menu.

3. Create a new widget, adjust all settings and get a ready-made shortcode.

Use Weather as a WordPress Shortcode



1. Create a new widget

You can use Elfsight Weather as a WordPress shortcode. To do this, you have to create a new widget, set all options in the Elfsight editor and save it.

2. Copy-paste the shortcode

You will see the new widget's shortcode in the widgets list. You can copy-paste it into the required place of your website.

Example of the Elfsight Weather shortcode:

[elfsight_weather id="1"]

Use Weather as a Gutenberg Block

You can use Elfsight Weather as a block for Gutenberg editor. Just add it to your page during page building and select one of the available Weather widgets from the list.

Use Weather as a WordPress Widget

To use Elfsight Weather as a WordPress widget, go to Appearance -> Widgets, and drag Elfsight Weather WordPress widget to any widget section. Next, select one of available Weather widgets from the list.

Use Weather as a Visual Composer Element

You can use Elfsight Weather as a Visual Composer element. Just add it to your page during page building and select one of the available Weather widgets from the list.

Options

Option Default Value Description
Content
Location (Zip code, City, Coordinates) New York Set a location to display weather for. The option supports geographical location (city, region), zip code and coordinates.
Temperature Unit °F Select how you'd like to display the temperature - Fahrenheit or Celcius.
Wind Speed Unit mph Select how you'd like to display wind speed.
Pressure Unit mph Select how you'd like to display pressure.
What to display Detailed Forecast Select the type of forecast you want to display.
Appearance
Background Themed Image Select background type - color or themed image.
Background Color rgb(77, 52, 243) Set background color for your widget.
Daily Forecast Background Color rgb(250, 250, 250) Set daily forecast background color.
Size 24 Set the widget size in pixels.
Enable Icons Animation ON Set this option to ON to activate weather icons animation.
Display temperature scale OFF Set this option to ON to display the temperature scale next to its value.
Expand forecast OFF Set this option to ON to expand the forecast list to make all days visible at once.
